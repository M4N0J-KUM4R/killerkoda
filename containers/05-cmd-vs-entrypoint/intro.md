# The Eternal Question

"What is the difference between CMD and ENTRYPOINT?"

This question has baffled Docker users for a decade. Today, we end the confusion.

We are going to use the **Bomb Defusual Analogy**:
-   **ENTRYPOINT** is the **Tool** you are holding (e.g., Wire Cutters). You *must* use this tool.
-   **CMD** is the **Target** (e.g., The Red Wire). It's the default thing to cut, but you can choose to cut something else.
