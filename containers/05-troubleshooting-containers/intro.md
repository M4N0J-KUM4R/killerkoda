# Crime Scene Investigation

A container has crashed. Users are screaming. The manager is pacing.

You need to find the root cause, and fast.

In this lab, you will learn the 3 tools of the Container Detective:
1.  **logs**: What did it shout before it died?
2.  **inspect**: Was it configured correctly?
3.  **exec**: Go inside and look around.

Let's solve the case.
